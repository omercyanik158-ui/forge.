# Phase 7 Known Limitations

- Manual device QA has not been executed in this environment.
- No automated performance benchmark threshold exists yet.
- Release-config checker validates app configuration and rollout flags, but not App Store / Play Store metadata.
- Data health check reports repairable issues but does not automatically repair them.
- Existing saved template programs remain readable even if the template engine flag is disabled.
- Physique adaptation persistence is separately gated and should be enabled only after product QA.
