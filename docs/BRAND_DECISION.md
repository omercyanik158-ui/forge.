# Release Identity

The current release identity is **FORGE**.

- App name: `FORGE`
- URL scheme: `forge`
- iOS bundle identifier: `com.yenispor.forge`
- Android package: `com.yenispor.forge`

These identifiers must be reviewed once more before the first App Store or
Google Play production build. After the first public release, changing the
bundle/package identifier creates a separate application and must not be done
as a routine rename.
